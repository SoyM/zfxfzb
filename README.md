#zfxfzb

####安装依赖

```
pip install -r requirements.txt
``` 

####提醒

* 进行开发调试时，建议将`HTTPCACHE_ENABLED` 设置为True
* `score.log` 文件是记录最后一次获取成绩的链接
* 在`run.py` 文件添加请求链接
* 有些是没有课程名称的，你们可以通过课程id获取课程名称


####运行

```
python run.py
```
